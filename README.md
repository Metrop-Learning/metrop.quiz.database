# metrop.quiz.database

The default quiz database of [Metrop](https://github.com/Metrop-Learning/Metrop).

## Database guideline

See the guideline at [https://metrop-learning.github.io/Tools-and-Documentation/](https://metrop-learning.github.io/Tools-and-Documentation/).

### Type of conflict :

| Code | Type |
|--|--|
| pos | Conflict of position (The border is recognized by both countries, but not its exact delimitation on the ground.) | 
| ter | Territorial Conflict (Dispute over the sovereignty of a specific region or island between recognized countries) |
| exi | Existential conflic (One of the countries (or its right to exist as an independent State) is not recognized by the other.) |
| abz | Active Buffer Zone (Conflict paused/frozen. The de facto border is secured by a UN peacekeeping force (buffer zone).) |
| war | War (Active invasion or occupation. The "de facto" frontline differs from the "de jure" official border) |   

use : `conflic_type` to add in a [] the list of conflic.  
use : `involved` to add in a [] the list of country involved.
